# csgo-match-forwarding
Forwarding csgo pro match record by qq bot


## 特性 | Feature

- [x] 自定义订阅战队
- [x] QQ机器人转发战报
- [x] 基于nonebot v1，方便插件移植
- [x] docker部署，易于上手

## 部署 | Deploy

支持两种方式的部署：**docker部署**和**手动部署**

> 如果你不熟悉nonebot或现有nonebot项目不是v1版本，可以考虑docker部署；
> 如果你已有nonebot v1项目，可以只移植插件文件和配置文件；
